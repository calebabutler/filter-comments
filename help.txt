./filter-comments.py -[flags]o [output] [inputs]
Flags include: 

  -h, --help:
    Gives this help screen.
  -k, --haskell-style
    Uses haskell style comments.
  -s, --shell-style
    Uses shell style comments.
  -l, --lisp-style
    Uses lisp style comments.
  -j, --java-style
    Uses java (C++ single line comments) style comments.
  -c, --c-style
    Uses c style comments.
  -o, --output
    Sets output file, if none defaults to stdout.
